## 运行方式
```bash
manim -pqh basic_shapes.py BasicShapes
```

把文件名和场景类名替换成对应项即可。

## 依赖
- manim Community Edition
- Python 3.10+
- 部分 3D / 视频输出场景可能需要 ffmpeg
